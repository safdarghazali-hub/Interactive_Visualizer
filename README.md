# 🌌 Interactive_Visualizer

This is an interactive art installation that transforms human movement and sound into a mesmerizing particle symphony. By leveraging Three.js and GLSL Shaders, the engine renders 80,000 live particles on the GPU, creating a fluid digital ether that reacts instantly to its environment.

The project is defined by three core pillars of interaction:

Kinetic Gesture Control: Utilizing MediaPipe for real-time hand tracking, the system allows users to sculpt the particle field in 3D space. A pinch scales the universe, a victory sign cycles through nine complex geometric patterns—from a shimmering DNA-Helix to a sprawling Galaxy—and a fist triggers a dramatic gravitational collapse into a central vortex.

Audio-Reactive Intelligence: Integrated with the Web Audio API, the system "listens" to the world. Low-end frequencies drive a rhythmic bass-pulse, while high-end treble causes individual particles to shimmer and spark, ensuring the visualizer is always in sync with the user's voice or ambient music.

Cyberpunk Aesthetic: The visual style is rooted in a "ghost-hybrid" aesthetic, utilizing UnrealBloom for neon radiance and Afterimage processing to create haunting motion trails. A toggleable Picture-in-Picture camera POV provides a mirrored "HUD" for precise control.

This Visualizer bridges the gap between the physical and the digital, turning the user into a conductor of light and geometry. It is a seamless blend of math, motion, and music, packaged into a high-performance web experience.

## 🎮 How to Interact
* **Pinch (Thumb & Index):** Scale the geometry.
* **Victory Sign (Peace):** Change to the next pattern.
* **Fist:** Collapse the structure into a black-hole vortex.
* **Audio Sync:** Click "Sync Audio" to make particles react to your microphone.
* Also has an Cam ON OFF live Feed Button
